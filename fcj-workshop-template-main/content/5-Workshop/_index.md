# Rookwork — Team Collaboration Management System on AWS Cloud

## Project Overview

**Rookwork** is an integrated team collaboration management software designed to run across multiple environments (browser and desktop app). The system uses a **Client-Server architecture** with a frontend built on **React 19**, a backend powered by **Spring Boot**, and the entire infrastructure deployed on **AWS Cloud** following the **High Availability (HA)** standard with multi-layered security.

This workshop guides you through deploying and operating the actual AWS infrastructure for the Rookwork project, including VPC network configuration, content delivery via CloudFront, WAF security, and integration of S3 storage and SES email services.
